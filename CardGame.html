Online Card Game

This is a simple card game that I made to demonstrate use of Unity's multiplayer tools. Specifically this project uses Game Sever Hosting, Lobby, Matchmaker, and Netcode for Game Objects.

The Basics

At its most basic, this game is just a simple turn-based card game where cards deal damage. Players can draw a card, use a card, and discard a card before ending their turn. Or they can skip their turn at any moment. At the moment the more interesting part of the game is how the networking is done and the tools used.

The Code

The game is built using the Unity game engine. All of the scripts are written in C# using Visual Studio (community). Being a one-man development team, I am responsible for all the game design, programming, art, and everything else. The bulk of the work so far has been on making the game a globally (probably best to play from north america) accessible multiplayer game using Unity's modern tools. I plan on sprucing the game up, but for now my focus has been on showcasing that I can work with client-server architectures and Unity's multiplayer tools.

The Networking

The in-game networking if you will (or the networking that actually updates player states and makes the game a game) is handled by Unity's Netcode for Gameobjects. This allows for RPCs as well as common data that one might want when networking and some automated syncing/state checking. I chose to use Netcode for Gameobjects over a third part solution, becuase Unity somewhat recently added a lot of tools and libraries for multi-player development in Unity. For this project I also decided to use Unity's Server Hosting, Lobby, and Matchmaking tools. All of these are part of the Unity Gaming Services offered by Unity. I created a dedicated linux server build for hosting my game and set up a fleet of servers using Unity's Game Server Hosting. I then used Lobby to group the players together and Matchmaker to handle matchmaking. The game is server-authoritative and follows a client-server architecture with clients passing inputs to the server to update the game state and propagate changes. There is still a lot that can be done with the multiplayer functionality, and I plan to make some updates in the future to clean up the lobby and matchmaking interfaces.

Play It Now

Click the link below to download the latest version of the game. This is a windows build. The game doesn't currently have a single player portion to it, and there isn't really a player base. This means that there probably won't be anyone else you can match up with in a game; however, if you want to test the game, you can run multiple instances on your machine, or you can get someone else to download it and find a match together. Also just a quick note, if you find that the "Find A Match" button keeps timing out, try a few times as the servers are not persistent (that would cost a fee while having them as needed does not).
