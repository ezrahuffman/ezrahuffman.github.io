Cal-Trans Mission

This was the first original game I made that I felt was worth putting on here. The original game was almost entirely finished in about 36 hours durring a game jam. I recently remade the game and put it on the Apple app store.

The Basics

"Potholes" drop from the top of the screen, and the player must click on them before the potholes reach the bottom of the screen. When a pothole reaches the bottom of the screen, the player loses a life.

Old Game Was On Google Play

After the game jam, I decided to finish squashing some bugs, and eventually I uploaded the game to the app store. The app is no longer active, because I don't keep it updated.

The Old Game

All of the coding was done using the Unity game engine. I wrote all of the scripts in C# using Visual Studio. At the time I was really new to making games, so the game is very simple. The potholes drop from the top of the screen, starting at one of four randomly selected spots. To make the game a little more challenging, the potholes also fall faster the longer the game goes on, though there is a max speed. This code was riddled with bugs, but generally worked. There were also limited options, sounds, and the game was overall not great.

The New Game

I remade the game from scratch over the course of about a week. I made the game more efficient (unneccessary object pooling). I made the UI dynamic, automatically resizing the UI to fit the screen size, and I added better sounds (Someone I know made the music for me) and settings.
