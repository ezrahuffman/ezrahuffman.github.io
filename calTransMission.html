Cal-Trans Mission

This was the first original game I made that I felt was worth putting on here. The game was almost entirely finished in about 36 hours durring a game jam. I did all of the programming and exporting. I did not do the sound or the art.

The Basics

"Potholes" drop from the top of the screen, and the player must click on them before the potholes reach the bottom of the screen. When a pothole reaches the bottom of the screen, the player loses a life.

Game Is On Google Play

After the game jam, I decided to finish squashing some bugs, and eventually I uploaded the game to the app store.

More About The Code

All of the coding was done using the Unity game engine. I wrote all of the scripts in C# using Visual Studio. At the time I was really knew to making games, so the game is very simple. The potholes drop from the top of the screen, starting at one of four randomly selected spots. To make the game a little more challenging, the potholes also fall faster the longer the game goes on, though there is a max speed.
