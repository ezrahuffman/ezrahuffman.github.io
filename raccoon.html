Raccoon Game

This is a 2D platformer that I made a while ago. After making a few small games, I decided it was time to make something more complex. The game is incomplete, but holds a special place in my heart.

The Basics

The raccoon game is a 2-D platformer where one plays as a raccoon lost in a strange world. The game is meant to tell a story of hardships and adventure, though currently, I am still working on the first level. Making this game taught me a lot about game development. The game is not close to being finished, but I hope to get back to it some day.

The Code

The game is built using the Unity game engine. All of the scripts are written in C# using Visual Studio (community). Being a one-man development team, I am responsible for all the game design, programming, art, and everything else. There are plans to outsource the sound in the future, but for now, it is all me.

Plans For the Future

As mentioned above, I am not actively working on the game at this moment. I hope to get back to it and build out some leveles in the future, but for now it is just something I look back on as a learning oprotunity that I am fond of. I would also like to point out the game is quite old at this point and is not representative of my current development skills.
