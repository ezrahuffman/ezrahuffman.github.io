Raccoon Game

This is a 2D platformer that I am still in the proccess of making. After making a few small games, I decided it was time to make something more complex.

The Basics

The raccoon game is a 2-D platformer where one plays as a raccoon lost in a strange world. The game is meant to tell a story of hardships and adventure, though currently, I am still working on the first level. This is a project I am passionate about, and one that I plan on making into the best game I can. To that point, I will be updating this website about once or twice a month with the newest build of the game.

The Code

The game is build using the Unity game engine. All of the scripts are written in C# using Visual Studio (community). Being a one-man development team, I am responsible for all the game design, programming, art, and everything else. There are plans to outsource the sound in the future, but for now, it is all me.

Plans For the Future

The game is still in the early stages of development, and I have a lot of work left to do, but there are definite plans for the future. There is not an exact number for how many levels will be in the final game, or how long the game will take to play through; however, I plan on having six stages in the final game. The current plan is to include unique mechanics in each stage, though that is subject to change. I am working on the game nearly every day and hope to create something I can be proud of, and something that others will enjoy.
